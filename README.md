# 🔐 SafeVault

SafeVault é um gerenciador de senhas desenvolvido em Python com foco em segurança da informação.
O sistema permite armazenar credenciais de forma segura utilizando criptografia, além de gerar senhas fortes automaticamente.

---

## 🎯 Objetivo

O projeto tem como objetivo oferecer uma solução simples para o problema do uso de senhas fracas e repetidas, promovendo boas práticas de segurança digital.

---

## ⚙️ Funcionalidades

* Armazenamento seguro de senhas
* Criptografia de dados sensíveis
* Geração de senhas fortes
* Interface gráfica simples para interação

---

## 🛠️ Tecnologias utilizadas

* Python
* Tkinter (interface gráfica)
* Cryptography (criptografia)
* SQLite (armazenamento de dados)

---

## ▶️ Como executar

1. Instale a biblioteca necessária:

```
pip install cryptography
```

2. Execute o programa:

```
python safevault_gui.py
```

---

## 🔐 Segurança

As senhas são armazenadas de forma criptografada, garantindo que mesmo em caso de acesso ao banco de dados, as informações não estejam em texto puro.

---

## 📌 Observações

Este projeto foi desenvolvido para fins acadêmicos, com foco em demonstrar conceitos de segurança, criptografia e desenvolvimento de software.
